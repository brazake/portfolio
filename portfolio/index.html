à propos de moi

Bonjour ! Je m'appelle Brandon. Je suis développeur web.
Auparavant, j’ai effectué plusieurs années en tant qu'Aide-soignant dans des hôpitaux généraux, psychiatrique et dans des maisons de retraite, ce qui m’a permis d’acquérir de très bonnes compétences telles que l’organisation, l’écoute, le conseil ainsi que des qualités relationnelles importantes. J’ai profité de mon opération du genou pour effectuer une reconversion professionnelle dans le Développement Web, domaine qui m’intéresse depuis de nombreuses années.

J’ai donc suivi la formation Développeur Web au sein d'une école de développement web en direct et à distance, O'clock. Je souhaite grâce à cette formation l’obtenir un Bac+2, puis trouvé un emploi pour continuer dans cet axe avec envie, motivation et des idées plein la tête.

Les services que je proposes

responsive design

Le Responsive Web design vise à l'élaboration de sites offrant une expérience de lecture et de navigation optimales quelle que soit la gamme d'appareil (téléphones mobiles, tablettes, ordinateur).

web design

Le web design vise organiser l’information dans les pages du site et adapter au mieux les contenus à ce support spécifique qu’est le web.

code propre

Un code qui respecte les règles et normes du langage de développement (HTML, CSS, PHP, Javascript, REACT )

référencement marketing

Englobe toutes les techniques visant à améliorer la visibilité d’un site web dans les résultats naturels des moteurs de recherche tels que Google, Yahoo ou Bing...
